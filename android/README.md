## Prerequisites

- Basic Java or Kotlin syntax
- Android Studio installed (latest stable)
- A device or emulator for testing

If you don't know Kotlin yet, spend one week on the [Kotlin official docs](https://kotlinlang.org/docs/home.html) before starting.

---

## How to Use This Guide

1. Follow levels in order — each builds on the previous.
2. Read the chapter, then type the code examples by hand (don't copy-paste).
3. Complete the mini project before moving to the next level.
4. Answer the interview questions at the end of each chapter without looking at the answers first.
5. Use the roadmap.md to track your daily progress.
